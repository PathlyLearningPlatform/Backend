# How to run

## Development

### Linux

- open project root directory in terminal of your choice
- run `./scripts/linux/start.local.sh`

### Windows

- open project root directory in terminal of your choice
- run `.\scripts\win\start.local.bat`

## Production

### Linux

- open project root directory in terminal of your choice
- run `./scripts/linux/start.sh`

### Windows

- open project root directory in terminal of your choice
- run `.\scripts\win\start.bat`
